## What is Screenie?

Screenie is a macOS screen recorder that automatically edits your recordings. Double-tap Control to record, double-tap to stop — Screenie adds smooth zoom on clicks, speeds up idle moments, renders a cursor with click highlights, and copies the result to your clipboard. No timeline, no video editor.

## Features

- **Double-tap Control** to start/stop recording
- **Auto zoom** on mouse clicks with spring physics
- **Speed ramping** — idle moments sped up 4-8x automatically
- **Cursor rendering** with smooth tracking and click ring highlights
- **Floating indicator** with recording timer
- **Preview HUD** — copy, open, or discard after recording
- **Clipboard ready** — paste your recording anywhere
- **Guided onboarding** — walks you through permissions on first launch

## Install

1. [Download Screenie.dmg](https://github.com/amywork777/Screenie/releases/latest/download/Screenie.dmg)
2. Open the DMG and drag Screenie to Applications
3. Launch Screenie — the onboarding will guide you through permissions

**Requirements:** macOS 14+ · Apple Silicon or Intel

## How it works

1. **Double-tap Control** → recording starts, floating pill shows timer
2. **Do your thing** → click, scroll, type — Screenie tracks everything
3. **Double-tap Control** → Screenie auto-edits (zoom, speed ramp, cursor) and copies to clipboard

## Build from source

```bash
git clone https://github.com/amywork777/Screenie.git
cd Screenie
open Screenie.xcodeproj
```

Build and run with Xcode. Requires macOS 14+ SDK.
